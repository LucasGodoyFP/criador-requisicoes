<h2 id="about"> Sobre</h2>

Aplicação desenvolvida em Python com Tkinter e Selenium para automação de pedidos do almoxarifado da Lab Imagem.

O sistema realiza a leitura de arquivos CSV, automatiza a criação de pedidos em sistema web e gera relatórios detalhados com status dos itens (processados, não encontrados ou ajustados).

A solução elimina tarefas repetitivas, reduz erros humanos e aumenta a produtividade no processo de requisição de materiais.

<h2 id="started"> Como rodar</h2>

<h3 id="prerequisites">Pré-requisitos</h3>

Requisitos necessários para rodar o projeto:

- [Python 3.10+](https://www.python.org/)
- Google Chrome

<h3 id="cloning" >Clonando</h3>

Como clonar o projeto

```bash
git clone https://github.com/LucasGodoyFP/criador-requisicoes.git
```

<h3 id="dependencies">Dependências</h3>

Instale as dependências necessárias:

```bash
pip install selenium
```

<h3 id="starting">Executando</h3>

Como executar o projeto

```bash
cd criador-requisicoes
python main.py
```
